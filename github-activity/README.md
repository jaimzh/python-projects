# GitHub Activity CLI

A simple command line interface to fetch and display recent activity of a GitHub user. It uses the GitHub API to get the data and shows it directly in your terminal.

Project idea from [roadmap.sh](https://roadmap.sh/projects/github-user-activity)

## Project Structure:

```
github-activity/
├── main.py        # CLI entry point and logic
└── README.md      # Project documentation
```

## Installation

make sure you have python 3 installed.

```bash
git clone https://github.com/LU347/github-activity.git
cd github-activity
```

## How to use it

you just need to provide the github username:

```bash
python main.py <username>
```

example:

```bash
python main.py octocat
```
